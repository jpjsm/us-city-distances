# README — Geofabrik Extracts

## Overview

This directory contains regional and national extracts of OpenStreetMap (OSM)
data obtained from Geofabrik GmbH. Geofabrik provides regularly updated,
quality‑controlled extracts of OSM’s global database, packaged by geographic
region. These extracts are used as the source for all OSM‑derived datasets in
this project.

## Contents

- **Raw extracts** in .pbf or .osm.bz2 format, downloaded directly from
Geofabrik.

- **Processed derivatives** generated by this project’s pipeline (e.g.,
filtered features, normalized geometries, or merged layers).

- **Metadata files** documenting download dates, bounding boxes, and checksums
for reproducibility.

## Usage Notes

Any public use of adapted or derivative databases must comply with the
Share‑Alike requirements of the ODbL.

When redistributing processed outputs, include attribution and a clear
indication of modifications.

For reproducibility, each extract should be referenced by its download date,
region, and original checksum.
