# U.S. Census Data

## Overview

This directory contains datasets sourced from the U.S. Census Bureau,
specifically the City and Town Population Totals (Vintage 2024) and related
public‑domain Census publications. These files provide authoritative
population counts and identifiers for all incorporated places in the United
States.

The data in this folder is used to construct the list of U.S. cities with
population ≥10,000, which serves as the foundation for the graph‑building and
adjacency computations in this project.

## Contents

- Raw Census files (e.g., Excel or CSV tables downloaded directly from the
Census Bureau)

- Processed datasets derived from the raw Census data

- LICENSE — explains the public‑domain status of Census data and the CC0
dedication for processed outputs

- ATTRIBUTION — courtesy citation for the Census Bureau as the original data
source

## Licensing

All raw Census data in this directory is a work of the U.S. federal government
and is therefore public domain under 17 U.S.C. §105.

All processed or transformed datasets produced from the Census data in this
project are released under the Creative Commons Zero (CC0 1.0) Public Domain
Dedication, allowing unrestricted reuse, modification, and redistribution.

A courtesy attribution to the U.S. Census Bureau is included, although it is not
legally required.
