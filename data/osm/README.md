# OpenStreetMap (OSM) road‑network data

## Overview

This directory contains datasets derived from OpenStreetMap (OSM) road‑network
data. These files include extracted road graphs, processed adjacency
relationships between cities, and any intermediate or final datasets generated
from OSM‑sourced geometry or topology.

The data in this folder forms the foundation for the city‑to‑city connectivity
and distance calculations used throughout the project.

## Contents

- Raw extracts of OSM road networks (when stored locally)

- Processed graph data generated from OSMnx or other OSM‑based tools

- Derived adjacency datasets representing city‑to‑city road connections

- LICENSE — full text of the Open Database License (ODbL 1.0)

- ATTRIBUTION — required attribution for OSM contributors
